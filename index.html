DayCal.io

My current project is DayCal.io, which is a clientside webapp that allows you to schedule your day. It's written mostly in Javascript, using D3.js and backbone.js. It pulls data from the Trello API and then allows you to import Trello cards into an interactive daily calendar view.

Experience

Web Design & Development

I'm experienced with both frontend design and backend development. I've used HTML/CSS quite extensively, along with Javascript and some of its derivatives (jQuery, backbone, D3), to develop for the web platform. I'm used to working with the mobile-first/progress enhancement paradigm.

I also have some backend development experience; I've written a platform for buying and selling items in Ruby on Rails for a class project and I've thrown together a few simple backends in PHP.

Game Development

Throughout high school, I created many plugins for a game called Minecraft. Using the Bukkit API, I wrote and contributed to about a dozen plugins, ranging from adding Harry Potter content to the game (e.g. spells, wands, etc) to fixing bugs present in the unmodified version of the game. Cumulatively, my projects have received over 300,000 downloads.

My experience as part of this community taught me the importance of open-source software and how to work collaboratively using Git. My hard work and professional attitude were recognized by the administrators of the Bukkit community, and I was chosen to join both of the staff teams: the Developer Staff team and the Forum Moderation team (both of which were extremely small subsets of the 350,000-member community). As part of the Dev Staff, I was required to decompile submissions and peruse the entire source for malicious or deceptive code. The thousands of submissions that I read provided lots of experience studying and understanding other people's code.

Desktop Development & Scripting

I've worked fairly extensively in Java and Python to create applications and scripts that serve a variety of purposes. I'm familiar with many of the language-specific features and have used numerous APIs/libraries in Java (e.g. Swing, Reflection, WolframAlpha) and Python (e.g Beautiful Soup). I'm also fairly well-versed in both C and MIPS (Assembly language).

Mobile Development

Mobile development is the area that excites me most because of the ease at which apps can reach millions of people. I love Apple's new Swift language and the iOS development workflow. I'm currently working on a SpriteKit game written entirely in Swift. I haven't ventured as far into Android development, but I'm familiar with many of the general ideas behind it and have a habit-tracking utility on the backburner.
